🧠 課題内容（提出必須）

課題①：図書管理システム（LibraryApp）を作ろう

仕様：

概要
簡易的な「図書管理アプリ」を作成します。
書籍(Book)を登録し、一覧表示し、削除できるシンプルなアプリです。
コマンドラインで操作できることを目指してください。

クラス設計例（必須条件）

クラス名	役割
Book	書籍1冊を表すクラス。タイトル・著者・出版年などをフィールドに持つ。
Library	書籍をリストで保持し、登録・一覧表示・削除などの操作を提供する。
LibraryApp	メインクラス。メニューを表示して、ユーザー入力を受け付ける。

機能要件
	1.	起動時に以下のメニューを表示する

図書管理システム
1: 書籍を登録する
2: 登録済みの書籍を一覧表示する
3: 書籍を削除する
0: 終了


	2.	書籍は タイトル・著者・出版年 を登録できること
	3.	ArrayList を用いて登録データを保持
	4.	登録した書籍を一覧表示できること
	5.	削除は「番号で指定」して行う（例：2番の本を削除）
	6.	適切なメソッド分割・クラス設計ができていること（例：addBook()、showBooks()、removeBook()など）

⸻

課題②（発展）

以下のうち、1つ以上を自由に追加しても構いません。
	•	書籍データをファイルに保存・読み込みできるようにする

	•	書籍に「貸出中/在庫あり」の状態を追加
	•	「検索機能」（タイトルの一部で検索）を実装

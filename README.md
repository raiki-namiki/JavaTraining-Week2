	•	プログラムの目的：
    書籍の管理や、利用者に書籍の登録・表示・消去などの機能を簡単に行うことができる図書管理システム。

	•	実行方法：
    javac LibraryApp.java → UTF-8 LibraryApp.java （文字化け対策） → Active code page: 65001 （文字化け対策）→ java LibraryApp

	•	クラス設計のポイント：
    Bookクラスでは、1冊の本の情報（タイトル・著者・出版年）を保持する。
    Libraryクラスでは、ArrayListを使用して複数のBookオブジェクトをリスト形式で保持し、登録（addBook）、表示（showBooks）、削除（removeBook）の具体的な処理を担当する。
    LibraryAppクラスでは、ユーザーへの画面表示や入力・入力されたデータに対するメソッドを実行する。

    •	追加機能：
    前回の課題では、一度例外のデータが入力されるとテキストを表示後に落ちてしまったため、今回は0から3以外の入力の場合に再び入力できるように改良した。また、数値入力の直後に文字列入力を行う際に発生するスキップ現象を防ぐため、nextLine() による空読み込み処理を追加し、スムーズに入力できるようにした。
